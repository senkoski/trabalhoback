Internet Segura: O que é Bloquear e Desbloquear Redes?

O que é bloquear uma rede?

Bloquear uma rede é como fechar uma porta para alguns sites ou aplicativos.

Assim, certas coisas ficam proibidas de entrar ou sair pela internet do lugar onde você está.

/div>

Por que alguém bloqueia a rede?

  • Para proteger contra vírus e coisas perigosas.
  • Para evitar distrações (como jogos ou vídeos na escola).
  • Para manter todo mundo seguro.

O que é desbloquear uma rede?

Desbloquear é abrir a porta de novo para aquilo que estava proibido.

Mas, se abrir sem permissão, pode ser perigoso!

Como posso evitar problemas com bloqueios e desbloqueios?

  • Não tente burlar bloqueios.
    (Usar truques pode deixar seu computador doente ou trazer problemas.)
  • Peça ajuda se precisar acessar algo importante.
    (Fale com um adulto, professor ou responsável pela internet.)
  • Cuidado com programas que prometem “liberar” a internet.
    (Eles podem ser perigosos!)
  • Use a internet só para o que é permitido naquele lugar.
  • Se não entender, pergunte!
    (Ninguém nasce sabendo tudo.)
Dica fácil para lembrar:
A internet é como uma escola ou uma casa:
Tem regras para todo mundo ficar seguro e feliz.
